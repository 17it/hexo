微信开发本地调试

发表于 2019-07-18 | 分类于 VUE JS | | 本文总阅读量 次

本次项目需要支持在微信中运作，由于在微信后台有配置可信域名，如果直接在微信里打开本地页面可能会导致某些诡异的错误，所以想法就是开发的时候在微信里访问 test-lebz.le.com ，然后代理到本地项目，这样就不用每次小修改都提交git，然后到测试服务器部署一遍。

现有条件
